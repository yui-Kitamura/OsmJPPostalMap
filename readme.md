# OSM郵便マップ
OpenStreetMapベースの日本郵便マップ表示・編集アプリ for Android

## アプリストア向け紹介文

「OSM郵便マップ」は、日本国内の郵便局や郵便ポストをOpenStreetMap (OSM) データに基づいて地図上に表示し、その情報を誰でも簡単に更新・追加できるアプリです。

### 主な機能
- **郵便マップ表示**: 現在地周辺の郵便局や郵便ポストをアイコンで分かりやすく表示します。
- **詳細情報の確認**: 収集時刻や窓口の営業時間の詳細を確認できます。
- **情報の編集・更新**: 古くなった収集時刻や営業時間をその場で修正し、OSMにアップロードできます。 ※OSMアカウントが必要です
- **新しいポストの追加**: 地図上にない新しいポストを見つけたら、その場で簡単に追加できます。 ※OSMアカウントが必要です
- **高度なフィルタリング**: 営業中の施設のみを表示したり、特定のサービスで絞り込むことが可能です。

みんなで情報を持ち寄り、より正確で便利な日本郵便マップを作り上げましょう！

## JavaRepositoryとしてのreadme

このプロジェクトは、Androidプラットフォーム向けのJavaで開発されたOpenStreetMapクライアントです。

### 技術スタック
- **言語**: Java
- **地図表示**: osmdroid
- **データ取得**: Overpass API
- **編集機能**: OSM API v0.6
- **アーキテクチャ**: MVVM (ViewModel, Repositoryパターンの採用)

### プロジェクト構造
- `app/src/main/java/.../ui/`: 画面（Activity）およびViewModel。
- `app/src/main/java/.../domain/`: ビジネスロジック、エンティティ、リポジトリのインターフェース。
- `app/src/main/java/.../data/`: API通信、キャッシュ、リポジトリの実装。
- `app/src/main/java/.../schedule/`: 収集時刻や営業時間のパース・解析ロジック。
- `app/src/main/java/.../core/`: 地図上のマーカーやダイアログなどの共通部品。
